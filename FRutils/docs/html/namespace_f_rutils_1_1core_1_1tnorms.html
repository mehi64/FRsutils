FRutils.core.tnorms.tn_minimum ( np.ndarray values) 
:return: In case of 1D array, returns minimum of the 1D array
In case of 3D array (nxnx2), performs elementwise minimum of each elemet of the nxnx2[0] array with nxnx2[1]. This is used for vectorized calculations of similarities.

:param values: 1D array or 3D array of form nxnx2
